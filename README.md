# WA-vjezbe3

**Pokretanje aplikacije Pizza_app**

_Backend - express_  
U terminalu otvoriti direktorij s backendom.  
Instalirati naredu npm install, te pokrenuti server naredbom **nodemon index.js**  
Backend će biti dostupan na adresi **http://localhost:3000**.

_Frontend - vue_  
U terminalu otvoriti direktorij s frontendom.
Instalirati naredbu npm install  
Pokrenuti aplikaciju naredbom **npm run dev** ili **vite**  
Frontend će biti dostupan na adresi **http://localhost:5173**.
